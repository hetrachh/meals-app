If you get 
Error: Reanimated 2 failed to create a worklet, maybe you forgot to add Reanimated's babel plugin?

step 1
delete all node modules and package-lock.json

step -2
go to babel.config 
add 
plugins: ["react-native-reanimated/plugin"],
under preset array 

step -3 
npm install

step -4 
expo start --clear
